Get started with Learning-Log in 2 minutes!

### 1. Open the App
Visit: [https://yourusername.github.io/learning-log/](https://yourusername.github.io/learning-log/)

### 2. Add Your First Book
- Click **"+ Add Book"** in the left sidebar
- Type the book name (e.g., "Atomic Habits")
- Click **"Add Book"**

### 3. Create Your First Note

#### Using Voice (Recommended for Hindi)
1. Select the book you just created
2. Click **"New Note"**
3. Select language: **English** or **‡§π‡§ø‡§Ç‡§¶‡•Ä**
4. Click **"Record"**
5. Click **"Start Recording"**
6. Speak your note
7. Click **"Stop Recording"**
8. Review and edit if needed
9. Click **"Save Note"**

#### Using Keyboard
1. Select the book
2. Click **"New Note"**
3. Select language
4. Click **"Type"**
5. Type your note
6. Click **"Save Note"**

### 4. Manage Your Notes
- **View**: Click any note to expand full text
- **Edit**: Click "Edit" in expanded note
- **Delete**: Click "Delete" in expanded note
- **Search**: Use search box at top
- **Sort**: Use dropdown to sort by date

### 5. Export to PDF
- Click **"Export PDF"** button in header
- Print dialog opens automatically
- Select **"Save as PDF"**
- Choose location and save

## üí° Tips

### For Best Results
- **Use Chrome** for Hindi voice recognition
- **Speak clearly** in a quiet environment
- **Review transcriptions** before saving
- **Export regularly** to backup your notes

### Keyboard Shortcuts
- `Ctrl+N` / `Cmd+N` - New note
- `Ctrl+S` / `Cmd+S` - Save note
- `Esc` - Close dialogs

### Language Switching
- Change language **before** creating a note
- You can have English and Hindi notes in the same book
- Each note remembers its language

## üîí Your Data

- Everything is stored **locally** in your browser
- **No account needed**
- **No data sent to servers**
